\# E-commerce Price Monitoring System



\## Tech Stack

\- Python

\- Selenium

\- Flask

\- MySQL



\## Features

\- Reads product names from CSV

\- Scrapes Amazon \& Flipkart prices

\- Stores data in MySQL

\- REST APIs for comparison



\## How to Run

1\. Install dependencies

2\. Configure MySQL

3\. Run `python app.py`
